🔧 VMware ESXi Setup on VMware Workstation

Building a nested virtualization environment for testing and training.

🧠 Overview

This lab focuses on installing and configuring VMware ESXi inside VMware Workstation to create a self-contained, virtualized environment for network testing and system performance validation.

🗺️ Topology

The setup includes one ESXi host, a management VM, and several virtual switches for isolated LAN and DMZ networks.

⚙️ Setup Steps

Follow these key steps to complete the setup:

  1. Install VMware Workstation and create a VM for ESXi.
  2. Enable nested virtualization and VT-x in settings.
  3. Attach three network adapters: LAN, DMZ, and Management.
  4. Deploy a pfSense firewall or test VM and validate connectivity.

💻 Example Commands


# Enable nested virtualization in PowerShell
Set-VMProcessor -VMName "ESXi-Lab" -ExposeVirtualizationExtensions $true

# Check ESXi management interface
ping 192.168.10.2

📊 Results

The environment successfully runs nested virtualization and simulates real enterprise scenarios, including firewall testing, network segmentation, and resource monitoring.
