First, open VMware Workstation and click on New Virtual Machine. Choose Custom (Advanced) and click Next. When prompted to select the hardware version, pick the latest one available for maximum compatibility.

Next, select Installer disc image file (ISO) and browse to locate the VMware ESXi ISO you downloaded earlier.

  • It is natively supported by ESXi.
  • It provides higher I/O efficiency in nested virtualization environments.
  • It reduces overhead between Workstation and the ESXi VM.

Continue to the disk configuration. Choose SCSI type and select Create a new virtual disk. For size, 60 GB is fine for testing, but 120 GB or more is ideal for larger setups or multiple nested VMs.
